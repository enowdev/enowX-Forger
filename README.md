# enowX-Forger

A developer tool for generating app icons across multiple platforms. Built with Tauri + SvelteKit.

## Features

- Generate icons for 35+ platforms (Tauri, Electron, Android, iOS, Next.js, PWA, Unity, etc.)
- Built-in image editor with layers support
- Background remover
- Icon discovery from Iconify
- Custom template support

## Development

```bash
bun install
bun run tauri dev
```

## Build

```bash
bun run tauri build
```
